# 隱私與資料處理說明 (範本)

本擴充套件會將使用者在 Canva 中觸發的「生成請求」（例如：簡報標題、提示詞、樣式選項）傳送至你所設定的後端伺服器，後端會呼叫 AI 服務生成內容與圖片，並把產出存放於雲端儲存（例如 GCS/S3）。

重要事項：

- 本擴充套件本身不會儲存或包含任何第三方 API 金鑰；所有敏感金鑰應由後端安全管理（Secret Manager）。
- 使用者輸入的提示詞與生成結果會經由後端處理，請在後端明確說明資料保留時長、是否用於模型訓練或分析等。建議預設短期保留（例如 30 天）。
- 若需收集額外使用者識別資訊（例如 email、帳號），請在此列明並取得使用者同意。

建議在 Chrome Web Store 上架時填入此檔內容並放置可公開連結，符合上架隱私政策要求。
